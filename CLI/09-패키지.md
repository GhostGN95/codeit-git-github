# 패키지 관지라

## APT
- 우분투 운영체제의 패키지 관리자
- apt update: 패키지 목록을 최신으로 업데이트한다
- apt search [패키지이름]
- apt list: 내가 설치한게 뭐지?
- apt install [패키지이름]
- apt upgrade [패키지이름]
- apt remove: 적당히 삭제
- apt purge: 삭제를 더 완벽하게

- gcc 설치 실전 연습
  - sudo apt update
  - sudo apt install build-essential -y

## Homebrew
- 맥 운영체계의 패키지 관리자
